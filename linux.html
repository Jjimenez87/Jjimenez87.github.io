mkdir makes a new directory.

[user@localhost]$ mkdir myproject

cd changes the current working directory.

[user@localhost]$ cd myproject

ls will list the files in the directory.

[user@localhost]$ ls

The Linux touch command creates empty files and modify the timestamps of a file.

[user@localhost]$ touch .gitignore

use the clip < command to copy the public key to our clipboard:

[user@localhost]$ clip < /Users/user/.ssh/id_rsa.pub
